# LocationGuard

SMS-based location tracker. Send an SMS trigger word to get GPS coordinates back. No internet needed on target.

## How It Works

1. Install on target phone
2. Set your number as admin + trigger word
3. Send trigger word via SMS
4. Receive Google Maps link back

## Permissions

READ_PHONE_STATE, ACCESS_COARSE_LOCATION, SEND_SMS, RECEIVE_SMS, RECEIVE_BOOT_COMPLETED
